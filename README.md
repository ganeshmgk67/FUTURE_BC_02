# FUTURE_BC_02

# Task 02 – ERC20 Token Creation

## Objective
To design, compile, and deploy a custom ERC-20 token using Solidity for learning and demonstration purposes.

## Description
A custom ERC-20 token named FutureToken (FTK) was created using OpenZeppelin’s ERC20 standard. The contract was compiled and successfully deployed on Remix VM (London). This task demonstrates basic smart contract development and deployment using Solidity.

## Tools Used
- Solidity
- Remix IDE
- OpenZeppelin ERC-20

## Note
This contract was deployed in a local Remix environment for educational purposes only and has no real monetary value.
